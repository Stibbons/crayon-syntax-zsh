=================
crayon-syntax-zsh
=================

This is a preliminary support for ZSH syntax highlighting for the Crayon Plugin for Wordpress.

Installation
============

 - send via FTP the ``langs/rust`` directory into ``wp-content/plugins/crayon-syntax-highlighter/langs/zsh``
   on your website

Urls
====

- Crayon Syntax Hightlighter : http://wordpress.org/plugins/crayon-syntax-highlighter/
- Wordpress: http://wordpress.org/
